Button group

Group a series of buttons together on a single line or stack them in a vertical column.
Wrap a series of buttons with .btn in .btn-group.

Sizing

Instead of applying button sizing classes to every button in a group,
just add .btn-group-* to each .btn-group, including each one when nesting multiple groups.

Collapse

Toggle the visibility of content across your project with a few classes.
The collapse JavaScript plugin is used to show and hide content.
Buttons or anchors are used as triggers that are mapped to specific elements you toggle.
Collapsing an element will animate the height from its current value to 0.
Given how CSS handles animations, you cannot use padding on a .collapse element.
Instead, use the class as an independent wrapping element.

  • .collapse hides content
  • .collapse.show shows content

How to use it?

It has been recommend to use a button with the data-bs-target attribute.
You can also use a link with the href attribute (and a role="button").
In both cases, the data-bs-toggle="collapse" is required.
