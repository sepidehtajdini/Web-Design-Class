متد ()SubStr

  • متد substr بخشی از یک رشته (string) را استخراج می‌کند.
  • این روش از یک موقعیت مشخص شروع می‌شود و تعداد مشخصی از کاراکترها را برمی‌گرداند
  • این متد رشته (string) اصلی را تغییر نمی‌دهد.
  • برای استخراج کاراکترها از انتهای رشته، از موقعیت شروع منفی استفاده کنید.
  • Syntax: string.substr(start, length)
  • پارامترها:
    1. شروع (Start):
      • این پارامتر ضروری است.
      • کاراکتر اول در index₀ است.
      • اگر Start بزرگتر از طول String باشد، SubStr عبارت "" را برمی‌گرداند.
      • اگر Start منفی باشد، SubStr از انتهای رشته حساب می‌شود.

    2. طول (Length):
      • این پارامتر اختیاری است.
      • در صورت حذف، بقیه‌ی رشته (String) را استخراج می‌کند.

    3. مقدار بازگشتی (Return Value):
      • یک رشته (String) است.
      • این رشته حاوی قسمت استخراج شده است
      • اگر طول 0 یا منفی باشد، یک رشته خالی برگردانده می‌شود.
  • چند مثال:
    let text = "Hello world!";
    let result = text.substr(0, 1);
    Result: H
    let result = text.substr(1, 4);
    Result: ello
    let result = text.substr(text.length-1, 1);
    Result: !
    let result = text.substr(-6, 6);
    Result: world!

متد ()Slice

  • متد Slice بخشی از یک رشته (String) را استخراج می‌کند.
  • این متد قسمت استخراج شده را در یک رشته جدید برمی‌گرداند.
  • Slice رشته (String) اصلی را تغییر نمی دهد.
  • پارامترهای شروع و پایان بخشی از رشته را برای استخراج مشخص می‌کند.
  • موقعیت اول 0 است، دومی 1 است، ...
  • اعداد منفی، از انتهای رشته شروع به انتخاب می‌کنند.
  • Syntax: string.slice(start, end)
  • پارامترها:
    1. شروع (Start):
      • این پارامتر ضروری است.
      • اولین کاراکتر 0 است.
    2. پایان (End):
      • این پارامتر اختیاری است.
      • موقعیت پایانی (تا، اما نه شامل).
      • پیش فرض طول رشته (String) است.
    3. مقدار بازگشتی (Return Value):
      • یک رشته (String) شامل بخش استخراج شده از رشته‌ی اصلی می‌باشد.
  • چند مثال:
    let text = "Hello world!";
    let result = text.slice(1, 4);
    Result: ell
    let result = text.slice(3);
    Result: lo world!
    let result = text.slice(-1);
    Result: !
    let result = text.slice(0);
    Result: Hello world!

متد ()SubString

  • متد ()substring کاراکترها را از میان دو کاراکتر از یک رشته استخراج می‌کند و رشته فرعی را برمی‌گرداند.
  • این متد کاراکترها را از شروع تا پایان استخراج می‌کند.
  • این متد رشته (String) اصلی را تغییر نمی‌دهد.
  • اگر شروع بزرگتر از پایان باشد، آرگومان‌ها جابجا می‌شوند: (4، 1) = (1، 4).
  • اگر مقادیر شروع و پایان کوچکتر از 0 باشند، همانند خود 0 عمل می‌کنند.
  • Syntax: string.substring(start, end)
  • پارامترها:
    1. شروع (Start):
      • این پارامتر ضروری است و بیانگر نقطه‌ی شروع است.
      • اولین کاراکتر در index₀ است.
    2. پایان (End):
      • این پارامتر اختیاری است.
      • موقعیت پایانی (تا، اما نه شامل).
      • در صورت حذف، بقیه‌ی رشته (String) را استخراج می‌کند.
    3. مقدار بازگشتی (Return Value):
      • رشته‌ای حاوی کاراکترهای استخراج شده است.
  • چند مثال:
    let text = "Hello world!";
    let result = text.substring(1, 4);
    Result: ell
    let result = text.substring(-3);
    Result: Hello world!
    let result = text.substring(0, 1);
    Result: H
    let result = text.substring(text.length - 1);
    Result: !

خلاصه

  1. متد ()substring شماره index شروع و پایان را دریافت می‌کند اما خروجی شامل ایندکس پایانی نمی‌شود.
  2. در متد ()substring اگر Start_Index بزرگتر از End_Index باشد، آرگومان‌ها
    خودبخود جابجا می‌شوند و در نهایت خروجی نمایش داده می‌شود اما متد ()slice چیزی در خروجی نشان نمی‌دهد.
  3. اگر از شماره ایندکس منفی ( مثلا ۱- ) در متد ()substring استفاده شود،
    برنامه آن را بعنوان عدد صفر درنظر می‌گیرد.
  4. متد ()Substr فقط شماره ایندکس شروع (و نه پایان) و سپس یک طول برای تعداد کاراکترهایی
    که قرار است استخراج شوند، دریافت می‌کند.
  5. متد ()Substr یک متد قدیمی می‌باشد و استفاده از آن پیشنهاد نمی‌شود.
    بهتر است از متد ()substring استفاده شود.
